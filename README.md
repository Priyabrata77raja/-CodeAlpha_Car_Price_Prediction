# -CodeAlpha_Car_Price_Prediction
🚗 Car Price Prediction

📌 Project Description

This project predicts the selling price of cars based on features like year, fuel type, kilometers driven, and more using Machine Learning techniques.

---

🛠️ Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

📊 Data Visualization

The following visualizations were created to understand the dataset:

- Histogram of Selling Price
- Boxplot: Year vs Selling Price
- Scatter Plot: Kilometers Driven vs Selling Price
- Boxplot: Fuel Type vs Selling Price
- Correlation Heatmap
- Scatter Plot: Actual Price vs Predicted Price

These visualizations helped in identifying trends and relationships between features.

---

⚙️ Steps Involved

1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Train-Test Split
6. Model Training (Regression Model)
7. Model Evaluation

---

📈 Model Performance

- Accuracy (R² Score): 82% on test data

---

🚀 Conclusion

The model is able to predict car prices with reasonable accuracy.
Visualization helped in understanding the impact of different features on price prediction.
